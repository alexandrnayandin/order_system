# Order System

Лабораторная работа №4 по дисциплине «Разработка кода информационных систем».

## Тема

Оптимизация проекта и разработка модульных тестов.

## Описание проекта

Проект представляет собой небольшой модуль расчета стоимости заказа интернет-магазина.  
В системе реализованы товары, позиции заказа, расчет скидки, расчет доставки и итоговой стоимости заказа.

## Используемые технологии

- C#  
- .NET 8  
- xUnit  
- Moq  
- coverlet.collector  
- ReportGenerator  
- GitHub Actions

## Возможности

* расчет суммы заказа;
* расчет скидки;
* расчет стоимости доставки;
* обработка ошибок;
* использование интерфейсов;
* модульное тестирование;
* mock-объекты через Moq;
* автоматический запуск тестов через GitHub Actions.
